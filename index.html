What is PicoOS?

PicoOS is an OS specifically made for Raspberry Pi Pico boards and processors

Supported devices and chips:

Raspberry Pi Pico

Raspberry Pi Pico W

Raspberry Pi Pico 2

Raspberry Pi Pico 2 W

Developer boards with an RP2040 or RP2350 chip

Installation

1. Install the PC-side dependencies

pip install mpremote psutil requests

2. Connect the Pico to your computer

3. Run the installer from the project root

python installer.py

4. Follow the prompts to flash MicroPython and copy PicoOS onto the board
