What is PicoOS?

PicoOS is an OS specifically made for Raspberry Pi Pico boards and processors

Supported devices and chips:

Raspberry Pi Pico

Raspberry Pi Pico W

Raspberry Pi Pico 2

Raspberry Pi Pico 2 W

Developer boards with an RP2040 or RP2350 chip

Features

Shell

Boots straight into a command shell instead of the raw MicroPython REPL вҖ” ls, cd, cat, mkdir, mv, mount, and a handful of system commands.

Apps

Apps are just .py files with an install() and main(path) function. Anything the shell doesn't recognize as built-in gets looked up in /apps.

Config

Status light, SD card pins, and the app registry are stored as JSON under /conf. Missing files regenerate with sane defaults instead of crashing.

Error logging

Failures get logged to the SD card if one's mounted, or to internal flash if not.

Installer

One command on your computer flashes MicroPython, copies the whole project over mpremote, and walks you through configuration.

Installation

1. Install the PC-side dependencies

pip install mpremote psutil requests

2. Connect the Pico to your computer

3. Run the installer from the project root

python installer.py

4. Follow the prompts to flash MicroPython and copy PicoOS onto the board
